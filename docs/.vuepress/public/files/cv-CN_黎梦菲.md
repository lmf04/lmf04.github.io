#### 3. 黄河口碳氮周转耦合驱动机制研究
* **全国大学生生命科学竞赛国家级三等奖**
* **2023年9月-2024年6月**
• 设计了跨三个生态带（淡水 R、河口混合 C 和近岸海水 M）的多变量因子实验，以模拟真实环境梯度。包含了温度变化（18°C, 4°C）、养分添加（NaNO₃）、微生物抑制（HgCl₂）和光照暴露，并进行了严格的平行重复。

• 整合实验数据，阐明了温度、微生物活动和养分输入如何共同调节河流-河口-海岸连续体中碳和氮的周转途径、速率和反馈机制。
